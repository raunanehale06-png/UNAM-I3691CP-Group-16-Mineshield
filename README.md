# UNAM-I3691CP-Group-16-Mineshield
Mineshield is a mobile safety app for hazardous workplaces. It uses smartphone sensors to detect risks like falls and noise, with cloud dashboards for monitoring. It tracks last known location underground, enabling fast emergency response without costly hardware.
